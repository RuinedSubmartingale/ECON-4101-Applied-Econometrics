# Discrete Random Variables
Let X = outcome of flipping a fair coin.
  X = 0 if heads
  X = 1 if tails
p_1 = P(X = 0) = 1/2
P_2 = P(X = 1) = 1/2

- `p_1` and `p_2` define Probability (mass/density) function
